# Инструкция по запуску Task Manager локально

## Проблема с npm offline режимом

Если у вас установлена переменная окружения `npm_config_offline=true`, npm не сможет загружать пакеты из интернета.

## Решение

### Вариант 1: Использовать готовые скрипты (РЕКОМЕНДУЕТСЯ)

1. **Двойной клик на `install.bat`** - установит все зависимости
2. **Двойной клик на `run.bat`** - запустит проект

### Вариант 2: Через командную строку (CMD)

Откройте **CMD** (не PowerShell!) и выполните:

```cmd
cd c:\Users\RAZER\Desktop\task-manager
set npm_config_offline=
npm install
cd backend
npm install
cd ..
npm run dev:all
```

### Вариант 3: Исправить переменную окружения Windows

1. Нажмите `Win + R`, введите `sysdm.cpl`, нажмите Enter
2. Перейдите на вкладку **"Дополнительно"**
3. Нажмите **"Переменные среды"**
4. В разделе **"Переменные пользователя"** или **"Системные переменные"** найдите `npm_config_offline`
5. Удалите эту переменную или установите значение в пустую строку
6. Перезапустите терминал

### Вариант 4: Через PowerShell (если CMD не работает)

```powershell
cd c:\Users\RAZER\Desktop\task-manager
[Environment]::SetEnvironmentVariable('npm_config_offline', $null, 'Process')
$env:npm_config_offline = $null
npm install --no-offline
cd backend
npm install --no-offline
cd ..
npm run dev:all
```

## После установки зависимостей

Проект будет доступен:
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:3001

## Проверка установки

Убедитесь, что созданы папки:
- `node_modules` в корне проекта
- `backend/node_modules` в папке backend

## Если ничего не помогает

1. Удалите все папки `node_modules` (если есть)
2. Удалите файлы `package-lock.json` (если есть)
3. Перезагрузите компьютер
4. Попробуйте установить зависимости заново

## Структура проекта

```
task-manager/
├── install.bat          # Скрипт установки зависимостей
├── run.bat              # Скрипт запуска проекта
├── package.json         # Зависимости Frontend
├── backend/
│   ├── package.json    # Зависимости Backend
│   └── .env            # Настройки Backend (уже настроено)
└── .env.local          # Настройки Frontend (уже настроено)
```
